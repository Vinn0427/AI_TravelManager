## 使用说明

1. 启动开发服务器：
```bash
cd travelmanager/frontend
npm install
npm run dev
```

2. 访问应用：
   - 打开 http://localhost:3000
   - 首次访问会跳转到登录页
   - 登录成功后进入首页

3. 测试流程：
   - 注册新用户 → 登录 → 查看首页 → 创建行程 → 查看个人中心

## 注意事项

1. 确保后端服务已启动（端口8080）
2. 确保数据库已创建并执行建表语句
3. 检查API接口地址是否正确配置（vite.config.js中的proxy）
4. Token会存储在localStorage中，清除浏览器数据会需要重新登录
